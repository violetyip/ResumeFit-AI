# 🎯 ResumeFit AI 智能简历匹配诊断引擎

基于大语言模型（LLM）的智能化求职辅助工具。彻底摒弃传统 NLP 向量模型的关键词硬匹配，采用前沿的 **"LLM as a Judge"** 架构，利用 DeepSeek 强大的上下文阅读与逻辑推理能力，精准发掘候选人的隐性潜力。

## 🌟 核心架构与亮点

- **🧠 纯大模型动态解析**：无需预设死板的技能词典，AI 根据岗位 JD 动态提取核心考点。
- **👁️ 语义概念降维打击**：突破行业黑话壁垒。模型能自动建立概念映射（如：识别简历中的 `U-Net` 即具备 `分割模型` 能力，识别 `OpenAI API` 即具备 `大模型应用` 潜力）。
- **💡 伯乐型包装建议**：针对未直接命中的技能，深挖候选人专业课程与项目基础，动态生成极具实操性的“简历包装”话术与高分建议。

## 🚀 极简部署与运行

1. **克隆项目**：
   ```bash
   git clone [https://github.com/你的用户名/你的仓库名.git](https://github.com/你的用户名/你的仓库名.git)
   cd 你的仓库名
   ```
2. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```
3. **启动服务**：
   ```bash
   streamlit run app.py
   ```

## 🔑 安全与隐私说明
本项目不收集、不存储任何简历数据及 API 密钥。**支持在网页前端界面直接填入并临时调用您的专属 DeepSeek API Key**，阅后即焚，保障个人资产与数据安全。
